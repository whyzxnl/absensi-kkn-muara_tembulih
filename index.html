📖 Panduan Setup Google Sheets
Ikuti langkah berikut untuk menghubungkan ke Google Sheets
Langkah 1 – Buat Google Sheets Baru Buka sheets.google.com dan buat spreadsheet baru.
Langkah 2 – Buka Apps Script Di Google Sheets, klik menu Ekstensi → Apps Script.
Langkah 3 – Tempel Kode Script Hapus semua kode yang ada, lalu tempel kode dari file google_apps_script.gs yang ada di folder ini.
Langkah 4 – Deploy sebagai Web App Klik Deploy → New Deployment → Web App. Set akses ke "Anyone". Salin URL yang muncul.
Langkah 5 – Tempel URL di Pengaturan Tempel URL tersebut di kolom URL di atas dan klik Simpan & Test Koneksi.
